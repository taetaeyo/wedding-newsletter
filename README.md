# 모바일 결혼 소식장 Next.js
결혼 소식을 알리기 위한 소식장 템플릿입니다.  

## 📚 내용 및 기능
- 결혼 일자 정보
- 사진첩
- 방명록
- 카카오톡 공유 기능 및 링크 공유 기능

## 🚘 시작하기
1. `$ cd {프로젝트 root 경로}` - 해당 프로젝트 폴더로 이동
2. `$ npm install` - 디펜던시 설치
3. `$ npm start` - 로컬로 실행
4. `$ npm run server` - json-server 실행(방명록)

## ❌ 오류 발생 시

`rm -rf package-lock.json` 
`rm -rf node_modules`
`rm -rf .next` 
후 다시 `npm install` 수행
